
# Evaluation and Review Work Samples

This repository contains selected, anonymized work samples demonstrating structured evaluation, document review, and written analysis developed through professional consulting experience.

## Focus Areas
- Review of written and visual material for accuracy, completeness, and consistency
- Application of defined standards, criteria, and requirements
- Identification of gaps, conflicts, and inconsistencies
- Clear, concise written findings and recommendations

## Background
My professional background is in technical consulting, where I routinely reviewed drawings, specifications, and system documentation against defined standards and operational requirements. This work required consistent evaluation, attention to detail, and clear written communication.

In addition, I have completed onboarding and task-based work on online platforms involving content review, comparison tasks, and structured evaluation using guidelines and scoring criteria.

## Repository Contents
- Document review sample
- Platform review experience summary

## Notes
All examples are generalized and do not contain proprietary or client-identifying information. These samples are intended to demonstrate approach, methodology, and communication style.
